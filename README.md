# 🛒 Customer Shopping Behavior Analysis

## 📌 Project Overview
This project analyzes customer shopping behavior using transactional data to uncover insights into spending patterns, customer segmentation, product preferences, and subscription trends. The goal is to support data-driven business decisions.

---

## 📊 Dataset Summary
- Total Records: 3,900
- Features: 18 columns
- Includes:
  - Customer demographics (Age, Gender, Location, Subscription Status)
  - Purchase details (Item, Category, Amount, Season, Size, Color)
  - Behavioral data (Discount usage, Purchase frequency, Review rating, Shipping type)

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- PostgreSQL
- Power BI

---

## ⚙️ Data Processing
- Handled missing values using median imputation by category
- Standardized column names (snake_case)
- Created new features like age groups and purchase frequency
- Loaded cleaned data into PostgreSQL for analysis

---

## 🧠 Key Analysis (SQL)
- Revenue analysis by gender
- High-spending customers using discounts
- Top-rated products
- Shipping type comparison
- Subscriber vs non-subscriber behavior
- Customer segmentation (New, Returning, Loyal)
- Repeat purchase and subscription trends
- Revenue contribution by age group

---

## 📈 Dashboard
An interactive Power BI dashboard was created to visualize:
- Customer segmentation
- Revenue trends
- Purchase behavior insights
- KPI metrics

---

## 💡 Business Insights
- Subscribers tend to spend more than non-subscribers
- Loyal customers contribute significantly to revenue
- Discount strategies need optimization
- Targeted marketing can improve engagement and sales

---

## 🚀 Conclusion
This project demonstrates end-to-end data analysis, from data cleaning and transformation to SQL-based analysis and dashboard visualization, showcasing practical data engineering and analytics skills.
